# Noise Analysis for Two-Port Networks: From First Principles to Optimization

This document provides a mathematically rigorous derivation of the fundamental theorems and equations governing noise in high-frequency two-port networks.

---

## 1. Mathematical Framework and Problem Definition

We consider a linear, time-invariant (LTI) network operating in a wide-sense stationary (WSS) thermal noise environment. All noise processes are assumed to be zero-mean, ergodic, and characterized by their power spectral densities (PSDs) within a narrow bandwidth $\Delta f$ around the frequency of interest.

We define:
*   $k$ as the Boltzmann constant.
*   $T$ as the absolute physical temperature (K).
*   $[S]$ as the scattering matrix of the network, normalized to a real characteristic impedance $Z_0$.
*   $[C_S]$ as the noise wave correlation matrix of the network.
*   $\Gamma_S$ as the complex source reflection coefficient.

---

## 2. Bosma's Theorem (Fundamental S-Parameter Representation)

### Theorem Statement
For a passive linear $N$-port network in thermodynamic equilibrium at temperature $T$, the internal noise generated by the network is completely characterized by its scattering parameters and physical temperature:

$$[C_S] = kT \left( [I] - [S][S]^\dagger \right) \quad \text{[Theorem: Bosma's Fundamental Result]}$$

where $[I]$ is the identity matrix and $\dagger$ denotes the conjugate transpose (Hermitian conjugate).

### Derivation from First Principles
1.  **Thermodynamic Equilibrium Constraints:**
    Assume the $N$-port is terminated at all ports by matched loads at the same physical temperature $T$. The system is in thermodynamic equilibrium. According to the Nyquist-Rayleigh-Jeans relation, each port receives an incident noise wave $a_i$ carrying an available power spectral density of:
    $$\frac{\langle a_i a_j^* \rangle}{\Delta f} = kT \delta_{ij} \implies \frac{\langle \mathbf{a} \mathbf{a}^\dagger \rangle}{\Delta f} = kT [I]$$

2.  **Linear Superposition of Waves:**
    At any port, the total outgoing wave vector $\mathbf{b}$ consists of the reflected/transmitted incident waves plus the internally generated noise wave vector $\mathbf{b}_n$:
    $$\mathbf{b} = [S]\mathbf{a} + \mathbf{b}_n$$
    Since the internal noise sources of the network are independent of the external terminations, the internal noise wave vector $\mathbf{b}_n$ is uncorrelated with the incident wave vector $\mathbf{a}$:
    $$\langle \mathbf{b}_n \mathbf{a}^\dagger \rangle = [0]$$

3.  **Detailed Balance Principle:**
    In thermodynamic equilibrium, the power outgoing from any port must equal the power incident on it to prevent net heat transfer:
    $$\frac{\langle \mathbf{b} \mathbf{b}^\dagger \rangle}{\Delta f} = \frac{\langle \mathbf{a} \mathbf{a}^\dagger \rangle}{\Delta f} = kT [I]$$

4.  **Algebraic Derivation:**
    Substitute the linear superposition relation into the detailed balance expression:
    $$\frac{\langle ([S]\mathbf{a} + \mathbf{b}_n)([S]\mathbf{a} + \mathbf{b}_n)^\dagger \rangle}{\Delta f} = kT [I]$$
    Expand the product:
    $$\frac{1}{\Delta f} \left( [S]\langle \mathbf{a}\mathbf{a}^\dagger \rangle[S]^\dagger + [S]\langle \mathbf{a}\mathbf{b}_n^\dagger \rangle + \langle \mathbf{b}_n\mathbf{a}^\dagger \rangle[S]^\dagger + \langle \mathbf{b}_n\mathbf{b}_n^\dagger \rangle \right) = kT [I]$$
    Applying the lack of correlation ($\langle \mathbf{b}_n \mathbf{a}^\dagger \rangle = [0]$) and the incident wave power ($\langle \mathbf{a} \mathbf{a}^\dagger \rangle = kT \Delta f [I]$):
    $$kT [S][S]^\dagger + [C_S] = kT [I]$$
    Isolating $[C_S]$ yields:
    $$[C_S] = kT \left( [I] - [S][S]^\dagger \right) \quad \square$$

## 3. Generalized Noise Temperature Equation

### Theorem Statement
The equivalent input-referred noise temperature $T_e$ of a two-port network connected to a source with reflection coefficient $\Gamma_S$ is expressed in quadratic form as:

$$T_e = \frac{[\alpha][C_S][\alpha]^+}{k(1 - |\Gamma_S|^2)} \quad \text{[Corollary: Derived Generalized Noise Temperature Equation]}$$

where $[\alpha]$ is a row vector mapping the device noise waves to the input, and $+$ denotes the conjugate transpose.

### Derivation from First Principles
1.  **Outgoing Noise Wave Formulation:**
    Let the noisy two-port network be represented by a noiseless network with S-parameter matrix $[S]$ and internal noise wave vector $\mathbf{b}_n = [b_{n1}, b_{n2}]^T$. 
    If a source with reflection coefficient $\Gamma_S$ is connected to Port 1 and Port 2 is matched ($\Gamma_L = 0$):
    $$a_1 = \Gamma_S b_1$$
    $$a_2 = 0$$

2.  **Wave Relations:**
    $$\begin{aligned}
    b_1 &= S_{11} a_1 + S_{12} a_2 + b_{n1} = S_{11} \Gamma_S b_1 + b_{n1} \\
    b_1 &= \frac{b_{n1}}{1 - S_{11}\Gamma_S}
    \end{aligned}$$
    The incident wave at Port 1 is:
    $$a_1 = \frac{\Gamma_S b_{n1}}{1 - S_{11}\Gamma_S}$$
    The outgoing noise wave at Port 2 is:
    $$b_2 = S_{21} a_1 + b_{n2} = \frac{S_{21}\Gamma_S}{1 - S_{11}\Gamma_S} b_{n1} + b_{n2}$$

3.  **Input Referral of Outgoing Wave:**
    To refer this noise wave to the input, we divide by the transmission gain term $\frac{S_{21}}{1 - S_{11}\Gamma_S}$:
    $$b_{n,\text{in}} = \frac{1 - S_{11}\Gamma_S}{S_{21}} b_2 = \Gamma_S b_{n1} + \frac{1 - S_{11}\Gamma_S}{S_{21}} b_{n2}$$
    Let the row vector $[\alpha]$ be defined as:
    $$[\alpha] = \begin{bmatrix} \Gamma_S & \frac{1 - S_{11}\Gamma_S}{S_{21}} \end{bmatrix}$$
    Then the equivalent input-referred noise wave is:
    $$b_{n,\text{in}} = [\alpha] \mathbf{b}_n$$

4.  **Available Noise Power Referral:**
    The mean-squared value of the input-referred noise wave is:
    $$\overline{|b_{n,\text{in}}|^2} = [\alpha] \langle \mathbf{b}_n \mathbf{b}_n^\dagger \rangle [\alpha]^\dagger = [\alpha] [C_S] [\alpha]^+ \cdot \Delta f$$
    Referring this to the available power at the input requires dividing by the source mismatch factor $(1 - |\Gamma_S|^2)$:
    $$P_{n,\text{in,avail}} = \frac{\overline{|b_{n,\text{in}}|^2}}{(1 - |\Gamma_S|^2) \Delta f} = \frac{[\alpha] [C_S] [\alpha]^+}{1 - |\Gamma_S|^2}$$
    Since the available thermal noise power from an equivalent source at temperature $T_e$ is $P_{n,\text{in,avail}} = k T_e$, we equate the expressions:
    $$k T_e = \frac{[\alpha] [C_S] [\alpha]^+}{1 - |\Gamma_S|^2} \implies T_e = \frac{[\alpha][C_S][\alpha]^+}{k(1 - |\Gamma_S|^2)} \quad \square$$

## 4. Haus-Adler Optimization Theorem (Cascade Ordering)

### Theorem Statement
For a cascade of two linear amplifier stages, Stage A and Stage B, with noise measures $M_A$ and $M_B$ respectively, the overall noise figure of the cascade is minimized if and only if the stage with the lower noise measure is placed first:

$$M_A < M_B \implies F_{AB} < F_{BA} \quad \text{[Theorem: Haus-Adler Optimization Theorem Result]}$$

### Derivation from First Principles
1.  **Definition of Noise Measure:**
    The noise measure $M$ of an amplifier stage with noise factor $F$ and available gain $G$ is defined as:
    $$M \triangleq \frac{F - 1}{1 - 1/G} \quad \text{[Definition]}$$

2.  **Cascade Noise Figure (Friis Formula):**
    For a cascade of Stage A followed by Stage B ($AB$), the total noise factor is:
    $$F_{AB} = F_A + \frac{F_B - 1}{G_A}$$
    For the reverse cascade of Stage B followed by Stage A ($BA$):
    $$F_{BA} = F_B + \frac{F_A - 1}{G_B}$$

3.  **Comparison and Inequality Manipulation:**
    We seek the condition under which the configuration $AB$ is superior to $BA$:
    $$F_{AB} < F_{BA}$$
    Substitute the Friis expressions:
    $$F_A + \frac{F_B - 1}{G_A} < F_B + \frac{F_A - 1}{G_B}$$
    Subtract $1$ from both sides to work with excess noise factors:
    $$(F_A - 1) + \frac{F_B - 1}{G_A} < (F_B - 1) + \frac{F_A - 1}{G_B}$$
    Group terms containing $(F_A - 1)$ and $(F_B - 1)$:
    $$(F_A - 1) - \frac{F_A - 1}{G_B} < (F_B - 1) - \frac{F_B - 1}{G_A}$$
    Factor out the terms:
    $$(F_A - 1) \left( 1 - \frac{1}{G_B} \right) < (F_B - 1) \left( 1 - \frac{1}{G_A} \right)$$
    Assuming active stages with gains $G_A > 1$ and $G_B > 1$, we divide by $\left(1 - \frac{1}{G_A}\right)\left(1 - \frac{1}{G_B}\right)$:
    $$\frac{F_A - 1}{1 - 1/G_A} < \frac{F_B - 1}{1 - 1/G_B}$$
    Applying the definition of noise measure:
    $$M_A < M_B \quad \square$$

## 5. Noise Figure of the Rothe-Dahlke Two-Port Model

### Theorem Statement
For a noisy two-port network driven by a source with impedance $Z_S = R_S + j X_S$ at temperature $T$, and represented by input-referred noise voltage $v_n$ and noise current $i_n$, the noise factor $F$ under the assumption of uncorrelated noise sources ($\overline{v_n i_n^*} = 0$) is:

$$F = 1 + \frac{\overline{v_n^2}}{4k T \Delta f R_S} + \frac{(R_S^2 + X_S^2)\overline{i_n^2}}{4k T \Delta f R_S} \quad \text{[Corollary: Final Derived Equation Result]}$$

### Derivation from First Principles
1.  **Rothe-Dahlke Circuit Representation:**
    A noisy linear two-port is represented as a noiseless two-port with two input-referred noise sources: a series voltage source $v_n$ and a shunt current source $i_n$.
    The source is modeled by its Thévenin equivalent: a noiseless source impedance $Z_S = R_S + j X_S$ in series with a thermal noise voltage source $v_S$, where:
    $$\overline{v_S^2} = 4 k T \Delta f R_S \quad \text{[Axiom: Nyquist Thermal Noise]}$$

2.  **Equivalent Loop Voltage:**
    Looking into the input port of the noiseless network, the total equivalent open-circuit noise voltage $v_{\text{total}}$ in the input loop is the superposition of the source noise, the two-port noise voltage, and the voltage drop across the source impedance due to the two-port noise current:
    $$v_{\text{total}} = v_S + v_n + Z_S i_n$$

3.  **Mean-Squared Voltage Evaluation:**
    Since the source noise $v_S$ is generated by a physically independent source, it is uncorrelated with $v_n$ and $i_n$. Therefore:
    $$\overline{|v_{\text{total}}|^2} = \overline{v_S^2} + \overline{|v_n + Z_S i_n|^2}$$
    Expanding the second term:
    $$\overline{|v_n + Z_S i_n|^2} = \overline{v_n^2} + |Z_S|^2 \overline{i_n^2} + 2\text{Re}\left( Z_S^* \overline{v_n i_n^*} \right)$$

4.  **Uncorrelated Approximation:**
    Assuming the input-referred voltage and current noise sources are uncorrelated ($\overline{v_n i_n^*} = 0$):
    $$\overline{|v_n + Z_S i_n|^2} = \overline{v_n^2} + (R_S^2 + X_S^2) \overline{i_n^2}$$

5.  **Noise Factor Derivation:**
    The noise factor $F$ is defined as the ratio of the total equivalent noise power (or mean-squared voltage) to the noise power from the source alone:
    $$F \triangleq \frac{\overline{|v_{\text{total}}|^2}}{\overline{v_S^2}} = 1 + \frac{\overline{|v_n + Z_S i_n|^2}}{\overline{v_S^2}} \quad \text{[Definition]}$$
    Substituting the uncorrelated expanded form and the thermal noise voltage of the source:
    $$F = 1 + \frac{\overline{v_n^2}}{4k T \Delta f R_S} + \frac{(R_S^2 + X_S^2)\overline{i_n^2}}{4k T \Delta f R_S} \quad \square$$

### Generalization (Correlated Noise Sources)
If the noise sources are correlated, we define the complex correlation admittance $Y_c = G_c + j B_c$ such that $i_n = i_u + Y_c v_n$, where $i_u$ is uncorrelated with $v_n$. The noise factor then generalizes to:
$$F = F_{\min} + \frac{R_n}{G_S} |Y_S - Y_{\text{opt}}|^2$$
where $R_n$ is the equivalent noise resistance, $Y_S = 1/Z_S$, and $Y_{\text{opt}}$ is the optimum source admittance.
